经过IE6&IE6+，chrome，firefox测试兼容使用方法： {yourId}替换成你的id

HTML

  1. <div id="{your_id}">
  2.     <div id="{your_id}_slides">
  3.         <div class="{your_id}_slide"><img src="{src}" /></div>
  4.         <div class="{your_id}_slide"><img src="{src}" /></div>
  5.         ...
  6.     </div>
  7.     <div id="{your_id}_text"></div>
  8.         <div class="{your_id}_text_block"></code>
  9.             <h1 class="{your_id}_text_title">标题</h1>
  10.             <p class="{your_id}_text_detail">内容</p>
  11.         </div>
  12.         ...
  13.     </div>
  14.     <div id="{your_id}_nav_left"></div>
  15.     <div id="{your_id}_nav_right"></div>
  16. </div>

CSS以下这些参数可以根据需要自行调整

  1. #{your_id}_wrapper{ width:100%; height:289px; background:#fff}
  2. #{your_id}{ width:800px; height:289px; margin:0 auto; position:relative; overflow:hidden}
  3. .{your_id}_slide{ width:800px; height:289px; float:left; overflow:hidden }
  4. #{your_id}_nav_left, #{your_id}_nav_right{ width:61px; height:61px; position:absolute }
  5. #{your_id}_nav_left{ background-image:url(lib/gallary_nav_left.png); left:700px; top:170px }
  6. #{your_id}_nav_right{ background-image:url(lib/gallary_nav_right.png); left:700px; top:60px }
  7. #{your_id}_text{height:289px; width:200px; background:url(lib/gallary_text_bg.png) repeat; position:absolute; left:0px; top:0px; color:#ccc; padding:20px; overflow:hidden;}
  8. #{your_id}_slides{ height:289px; position:absolute; top:0; left:0}
  9. #{your_id}_text h1{ font-size:17px; margin-bottom:8px; text-align:left; font-weight:bold; width:200px; height:27px; overflow:hidden }
  10. #{your_id}_text p{ font-size:14px; line-height:27px; text-indent:30px}
  11. .{your_id}_text_block{display:none}

JS

  1. <script type="text/javascript" src="lib/jquery-1.9.1.js"></script>
  2. <script type="text/javascript" src="lib/jquery.mousewheel.js"></script>
  3. <script type="text/javascript" src="jquery.gallary.js"></script>

调用

  1. $("#{yourId}").gallary({
  2.         "pic_nums" : 5, // 图片个数,对应上述html里IMG标签个数；默认3个
  3.         "hide_text" : false, //是否自动隐藏文字；默认是true，即自动隐藏
  4.         "auto" : true, //是否自动播放；默认是false
  5.         "auto_interval": 3000, //自动播放的间隔，单位毫秒；默认3000毫秒，即三秒
  6.         "mouse" : true, //是否支持鼠标切换图片；默认是false
  7.         "mouse_id" : this.attr("id") //鼠标切换触发的id；默认是主id即{your_id}
  8. });
